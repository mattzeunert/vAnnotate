vAnnotate

Annotate JS files with the values of local variables from when the program was running.

vAnnotate logs variable assignment, function calls etc during execution and then overlays the source code with the collected data.

For example the underscore.js test suite runs this code to find even numbers:

_.filter([1, 2, 3, 4, 5, 6], function(num){ return num % 2 == 0; });

vAnnotate on Github



Install vAnnotate

vAnnotate is on npm:

sudo npm install -g vannotate

Then run vAnnotate in a folder containing a static website. The directory will be served at localhost:4000 and when you open an html file the Javascript code it loads will be annotated.

You can see annotated files at localhost:4000/vAnnotate/results.
