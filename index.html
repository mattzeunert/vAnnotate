vAnnotate

vAnnotated annotates Javascript files with the values of local variables from when the program was running.

See an example of an annotated file

vAnnotate on Github

How it works

vAnnotates requires a static html page that loads external Javascript code.

The vAnnotate command (installable using `npm install vAnnotate -g`) then creates a local server that can be used to open the html file (At http://localhost:3000/filename.html). When the browser requests the referenced Javascript code vAnnotate injects logging code to keep of variable assignments etc.

Then the annotated files can be viewed at http://localhost:3000/vAnnotate/results/
